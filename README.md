  Tic Tac Toe Game

A modern and responsive **Tic Tac Toe** game built using HTML, CSS, and JavaScript.  
This project features a clean UI, smooth gameplay, active player highlighting, and restart functionality.

  Features

-  Classic Tic Tac Toe gameplay  
-  Modern UI with gradient background  
-  Two-player mode (Player O vs Player X)  
-  Active player highlight animation  
-  Winner detection with message display  
-  Restart game button  
-  Fully responsive design for mobile & desktop  

 Tech Stack

- HTML5
- CSS3
- JavaScript
